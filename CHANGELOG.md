## [Unreleased]

### Added

- Three-tier token architecture (global, semantic, component) with JSON source files
- Custom token build pipeline — resolves references and outputs CSS custom properties
- Light and dark theme token sets
- Global tokens: colors, spacing, typography, radii, shadows
- Semantic tokens: text colors, background colors, border colors, interactive states
- Component tokens: button variants (primary, secondary, danger, disabled)
- Project scaffolding with TypeScript strict mode, tsup build, vitest
